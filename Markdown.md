# 1. Title
> ### Two format 
>> Using = or -  

 #### Example:  
 ```
 This is the first level title!
 =
 This is the secondary title!
 -
 ```
 #### Effects:  
 This is the first level title!
 =
 This is the secondary title!
 -  

> Using #  

The number of "#" represents the level of the title  
#### Example:  
```
# First level
## Second level   
...
###### Five level
```
#### Effects:  
# First level
## Second level   
...  
##### Five level
